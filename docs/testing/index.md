# Testing

VibePlugins test utilities for RocketMod plugin commands, events, entities, and multi-step scenarios.

| Topic | Description |
|---|---|
| [Command Testing](../command-testing.md) | Fluent builder API for executing and asserting plugin commands |
| [Event Testing](../event-testing.md) | Monitor and assert server events during test execution |
| [Mock Entities](../mock-entities.md) | Create players, zombies, and animals with configurable properties |
| [Scenario Builder](../scenario-builder.md) | Given/When/Then multi-step test scenarios |
